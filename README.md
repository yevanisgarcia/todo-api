# To-Do API (FastAPI)

This is a REST API built with FastAPI and SQLite.

## Features
- Create, Read, Update, Delete TODOs
- Toggle completion status
- Filtering and pagination
- Timestamps (created_at, updated_at)

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite

## Run locally

```bash
uvicorn app.main:app --reload
